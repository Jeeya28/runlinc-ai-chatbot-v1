# Runlinc AI Chatbot

A simple AI chatbot created using HTML, CSS and JavaScript.

## Features

- Simple chatbot interaction
- Keyword based responses
- Chat interface
- Enter key support
- Lightweight and beginner friendly

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone the repository

git clone https://github.com/USERNAME/runlinc-ai-chatbot.git

2. Open the project folder

3. Open `index.html` in your browser

## Example Commands

You can type:

- hello
- how are you
- who are you
- help
- bye
